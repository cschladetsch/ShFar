# Demo Workspace

Version: v1.0.0

This demo folder exists so you can safely learn `far` before running it on a
real project. It intentionally contains:

- `OldClass` rename examples in C++ files
- banner comments like `// -- Parser implementation -----`
- a `recieve` typo
- a `mylib_static` CMake target

If you want to reset changes after experimenting:

```bash
git restore demo/
```
